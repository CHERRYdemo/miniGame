# 圣诞树魔法世界 - Windows 使用指南

## 快速开始

1. **双击 `开始游戏.bat`** 文件
2. 如果提示需要 Python，运行 `setup_python.bat` 自动设置
3. 等待浏览器自动打开（约 3 秒）
4. 开始游戏！

## 系统要求

- **操作系统**: Windows 10 或更高版本
- **浏览器**: Microsoft Edge（推荐）或其他现代浏览器
- **运行环境**: Python 3（可选，游戏自带便携式 Python）

## 文件说明

- `开始游戏.bat` - **主启动文件**，双击即可运行
- `setup_python.bat` - Python 环境设置脚本（首次使用或提示需要时运行）
- `检查环境.bat` - （可选）环境诊断工具，用于故障排除
- `使用说明.txt` - 详细使用说明和故障排除

## 常见问题

### 问题1: 提示"未找到运行环境"

**解决方案**: 安装 Python 3
1. 访问 https://www.python.org/downloads/
2. 下载并安装 Python 3
3. **重要**: 安装时务必勾选 "Add Python to PATH"

### 问题2: 浏览器没有自动打开

**解决方案**: 手动打开浏览器，访问显示的地址
- 通常是: `http://localhost:8080/christmas-1/index.html`

### 问题3: 游戏加载很慢

**原因**: 首次运行需要从网络下载资源

**解决方案**:
- 确保网络连接正常
- 首次加载会从网络下载资源，之后会自动缓存

### 问题4: 页面显示"无法访问"

**解决方案**:
1. 确保防火墙允许 Python 访问网络
2. 检查是否有其他程序占用了 8080 端口
3. 尝试关闭后重新运行脚本

## 游戏玩法

1. 点击 **"🎄开始"** 按钮开始堆圣诞树
2. 对着摄像头做 **"O" 型嘴**吐泡泡
3. 泡泡会累积成圣诞树
4. 完成后，做 **✊握拳 -> ✋张开** 手势，泡泡会变成粒子圣诞树
5. 点击圣诞树可以进入第二颗树页面

## 停止游戏

在服务器窗口按 **Ctrl+C** 即可停止

## 注意事项

- 首次运行需要网络连接来加载资源
- 建议使用 Edge 浏览器以获得最佳体验
- 需要允许浏览器访问摄像头权限
- 如果遇到问题，先运行 `检查环境.bat` 检查系统环境

## 技术支持

如果遇到问题：
1. 先运行 `检查环境.bat` 检查环境
2. 查看 `使用说明.txt` 获取详细帮助
3. 确保 Python 已正确安装或运行 `setup_python.bat` 设置便携式 Python
