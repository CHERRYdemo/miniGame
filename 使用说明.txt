========================================
    圣诞树魔法世界 - 使用说明
========================================

【快速开始 - 最简单方式】
1. 双击 "开始游戏.bat" 文件
2. 如果提示"未找到运行环境"，运行 "setup_python.bat" 自动设置 Python
3. 等待浏览器自动打开（约 3 秒）
4. 如果浏览器没有自动打开，请手动访问显示的地址

【系统要求】
- Windows 10 或更高版本
- Microsoft Edge 浏览器（推荐）或其他现代浏览器
- Python 3（游戏自带便携式 Python，无需安装）

【如果遇到问题】

问题1：提示"未找到运行环境"
解决（推荐）：运行 "setup_python.bat" 自动设置便携式 Python
  - 脚本会自动下载 Python 到本地
  - 完成后即可使用，无需安装到系统
  
解决（备选）：安装系统 Python 3
  - 访问 https://www.python.org/downloads/
  - 下载并安装 Python 3
  - 安装时务必勾选 "Add Python to PATH"

问题2：浏览器没有自动打开
解决：手动打开浏览器，访问显示的地址（通常是 http://localhost:8080/christmas-1/index.html）

问题3：页面显示"无法访问"
解决：
  - 确保防火墙允许 Python 访问网络
  - 检查是否有其他程序占用了 8080 端口
  - 尝试关闭后重新运行脚本

问题4：游戏加载很慢
解决：
  - 首次运行需要从网络下载资源，请耐心等待
  - 确保网络连接正常
  - 这是正常现象，资源会自动缓存到浏览器

【游戏玩法】
1. 点击 "🎄开始" 按钮开始堆圣诞树
2. 对着摄像头做 "O" 型嘴吐泡泡
3. 泡泡会累积成圣诞树
4. 完成后，做 ✊握拳 -> ✋张开 手势，泡泡会变成粒子圣诞树
5. 点击圣诞树可以进入第二颗树页面

【停止游戏】
在服务器窗口按 Ctrl+C 即可停止

【注意事项】
- 首次运行需要网络连接来加载资源
- 建议使用 Edge 浏览器以获得最佳体验
- 需要允许浏览器访问摄像头权限

========================================
如有问题，请检查：
1. Python 是否正确安装或配置
2. 浏览器是否支持现代 Web 标准
3. 防火墙是否阻止了本地服务器
