使用说明 - 圣诞树互动网页
============================

重要提示：
---------
由于浏览器的安全限制，不能直接双击 index.html 文件运行。
必须使用本文件夹中的"启动网页.bat"来启动。

Windows 用户使用方法：
---------------------
1. 双击运行 "启动网页.bat" 文件
2. 等待几秒钟，浏览器会自动打开网页
3. 如果没有自动打开，请在浏览器中访问：http://localhost:8080
4. 使用完毕后，关闭命令窗口即可停止服务器

Mac/Linux 用户使用方法：
----------------------
1. 打开终端
2. 进入本文件夹
3. 运行：chmod +x 启动服务器.sh
4. 运行：./启动服务器.sh
5. 在浏览器中访问：http://localhost:8080

常见问题：
---------
Q: 双击"启动网页.bat"后提示"未找到服务器"？
A: 您的电脑需要安装 Python 3。请访问 https://www.python.org/downloads/ 下载安装。
   安装时请务必勾选 "Add Python to PATH" 选项。

Q: 浏览器没有自动打开？
A: 请手动在浏览器地址栏输入：http://localhost:8080

Q: 手势识别不工作？
A: 请确保：
   - 使用"启动网页.bat"启动，而不是直接打开 index.html
   - 浏览器允许访问摄像头
   - 摄像头正常工作

Q: 如何停止服务器？
A: 关闭黑色的命令窗口即可。

文件说明：
---------
- 启动网页.bat      Windows 启动脚本（双击运行）
- 启动服务器.sh     Mac/Linux 启动脚本
- index.html        主网页文件（不要直接双击）
- assets/           网页资源文件
- photos/           照片文件夹

技术支持：
---------
如果遇到问题，请检查：
1. 是否使用"启动网页.bat"启动（不是直接打开 index.html）
2. 浏览器控制台是否有错误信息（按 F12 打开）
3. 是否允许浏览器访问摄像头权限
